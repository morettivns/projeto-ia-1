Esse é o primeiro projeto que posto em meu GitHub.
Por meio desse projeto consegui desenvolver uma aplicação em Python que demonstra uma inteligência artificial funcionando seja em um video retornado pela mesma ou em uma câmera ao vivo.
Utilizei o modelo YOLO para a detecção de objetos em imagens em conjunto com a interface gráfica Custom Tkinter.

O sistema é simples, fácil de usar e bem funcional, podendo conectar-se com sua webcam enquanto o modelo retorna imagens em tempo real por meio da interface.

Funcionalidades:
Captura de video em tempo real, Detecção de objetos utilizando modelo YOLO, Navegação entre janelas, Processamento em tempo real, Demonstração do modelo em videos.

Tecnologias Utilizadas:
Python 3, OpenCV, Ultralytics YOLO, CustomTkinter, Pillow e Pathlib.
